Accesserty UI Kit Checkbox

How to use

HTML

  1. Copy file (checkbox.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-checkbox> tag

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

FAQ

How does it compare to native checkboxes?

Using the au-checkbox ensures that screen readers correctly interpret the association between the label and input, making it recommended for use.

But even so, this is still not enough. If there is a group of multiple related checkboxes, it's best to use role="group", aria-labelledby, <ul> and <li>. This way, it is possible to know how many checkboxes are in the same group.
