Accesserty UI Kit Card

How to use

This card module primarily restricts the DOM order of the card content, with all other styles being customizable by the user.

HTML

  1. Copy file (card.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-card> tag

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

Slot

Name Description
heading Set the title block of card
media Set image or video
content Set main description
footer Set the footer of card

Demo

Be mindful of the order in which you write elements and the final DOM order. The au-card enforces that the heading must appear first, allowing screen reading software to logically read the title before the content.

You can still use grid or flex to visually swap positions.(set &::part(au-card-container) display)
