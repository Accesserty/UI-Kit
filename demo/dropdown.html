How to use

  1. Copy file (dropdown.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-dropdown> and <au-dropdown-item> tags

Event listeners

The <au-dropdown> component dispatches a selected event when an item is clicked.

document.querySelector('au-dropdown').addEventListener('selected', (e) => { console.log('Selected value:', e.detail.value); });

Conclusion

The AuDropdown component ensures that users can interact with menus using standard keyboard patterns (Arrows, Enter, Space, Escape) while providing clear semantic hints to assistive technologies.
