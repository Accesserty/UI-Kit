Accesserty UI Kit Breadcrumbs

How to use

HTML

  1. Copy file (breadcrumbs.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-breadcrumbs> tag

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

Attribute

The Accesserty UI Kit breadcrumbs inherits attributes, with some custom options available:
Name Description Type Default value
id Sets id string
class Sets classname string
aria-label Sets aria-label string
separator Sets the separator between breadcrumb items. string /
items Sets the text and url path of breadcrumb item. { "text":string, "url":string }

Conclusion

This is a simple Web Component with basic functionalities. It includes:

  • Inheritance of id, class, and aria-label.
  • Addition of a separator to change the linking symbol and items for data handling.
  • Each item can prepend a slot to change its icon.
  • Indicates the current page with aria-current="page".
  • Uses CSS Variables to adjust the main appearance.

For developers, creating an accessible breadcrumbs web component is not difficult, and its application can be very convenient, making it a worthwhile feature to develop or use.
