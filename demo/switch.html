Accesserty UI Kit Switch

How to use

HTML

  1. Copy file (switch.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-switch> tag

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

FAQ

Why use input[type="checkbox"] instead of creating everything with div elements?

We don't need to use aria-checked and tabindex attributes. Whenever possible, it's best to use native HTML elements.
