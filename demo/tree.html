Accesserty UI Kit Tree

How to use

  1. Copy file (accordion.js or accesserty-ui-kit.js or accesserty-ui-kit.min.js)
  2. Add the JavaScript file to HTML using the <script src="" defer> tag
  3. Use the <au-tree> tag

Ensure to replace 'your-path' with the actual path to your JavaScript file. Also, make sure the path and filenames are correct to avoid loading errors.

Basic

Checkbox Tree

Supports tri-state selection (Indeterminate).

Custom Toggle Label

Customize the aria-label of toggle buttons.

const customLabelTree = document.getElementById('tree-custom-label'); customLabelTree.toggleLabel = (node) => { return `Expand or collapse ${node.label}`; };
